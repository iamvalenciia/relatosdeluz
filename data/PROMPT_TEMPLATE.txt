========================================
TÉCNICA DE STORYTELLING DE ALTO ENGANCHE
========================================
El objetivo es que el espectador NO PUEDA dejar de ver. Cada segundo debe tener tensión, emoción o revelación.

ESTRUCTURA NARRATIVA (Arco de 3 actos en 2-3 minutos):

1. HOOK IRRESISTIBLE (0-10s): Los primeros 10 segundos son CRÍTICOS.
   - Abre con una AFIRMACIÓN IMPACTANTE o un MOMENTO DE TENSIÓN.
   - NO uses preguntas genéricas. Usa afirmaciones que generen curiosidad.
   - El espectador debe pensar "¿Qué? Necesito saber más."
   - Ejemplo: "En 1839, un profeta moribundo se levantó de su cama y caminó hacia el río. Lo que hizo después cambió la historia de la Iglesia para siempre."

2. DESARROLLO CON TENSIÓN CONSTANTE (10s-2:00):
   - Cada 10-15 segundos debe haber un "micro-hook" o cambio emocional.
   - Usa técnicas de storytelling:
     * "Pero lo que no sabía era que..."
     * "Y entonces sucedió algo inesperado..."
     * "La respuesta llegó de la forma más improbable..."
   - Construye TENSIÓN EMOCIONAL REAL: ¿qué estaba en juego? ¿qué podía perder?
   - Muestra la HUMANIDAD del personaje: el miedo, la duda, el dolor, la fe.
   - Usa ESCRITURAS ESPECÍFICAS: cita versículos, referencias concretas.

3. CLÍMAX EMOCIONAL (2:00-2:30): El momento donde todo cambia.
   - La respuesta de Dios, el milagro, la revelación, la bendición.
   - Conecta con el espectador: "Esto no fue solo para [personaje]. Es para ti."
   - Declaración de doctrina pura, no opinión. Evangelio real.

4. CIERRE REFLEXIVO (2:30-3:00): Frase final completa que resuena.
   - Conecta la historia con una verdad eterna y universal.
   - Deja al espectador sintiendo algo profundo.
   - La frase debe ser COMPLETA, no cortada.

REGLAS DE ENGANCHE:
- NUNCA empieces con "Hola", "Bienvenidos", o saludos.
- NUNCA termines con "Gracias", "Suscríbete", o despedidas.
- NUNCA uses "..." al inicio ni al final. Las frases deben ser COMPLETAS.
- NUNCA hagas preguntas al inicio o al final.
- Cada transición debe tener un "gancho" que mantenga la atención.

AUDIO TAGS (ElevenLabs V3)
========================================
Usa estos tags con INTENCIÓN EMOCIONAL, no mecánicamente. Cada tag debe tener una razón clara de por qué está ahí.

Tags disponibles:
- [softly] → Para momentos íntimos, sagrados, revelaciones personales. Como si bajaras la voz para compartir un secreto sagrado.
- [pause] → Silencio dramático ANTES de una verdad importante. Da peso y gravedad a lo que viene después.
- [pensive] → Para reflexión profunda, cuando el narrador procesa algo que lo conmueve.
- [warmly] → Para conexión personal directa con el espectador. Como si le hablaras a un amigo.
- [hopeful] → Para el clímax positivo, la resolución, la promesa de Dios cumplida.
- [awe] → Para momentos de asombro genuino ante la grandeza de Dios o un milagro.

TAGS PROHIBIDOS:
- [reverently] → NO USAR. Distorsiona la voz del modelo. PROHIBIDO.

REGLAS DE USO:
- Máximo 12 tags por video (más largo = más tags permitidos)
- NUNCA pongas dos tags seguidos ([softly] [pause] → INCORRECTO)
- El tag va ANTES de la frase que modifica
- [pause] siempre va solo, separando dos ideas. Nunca combinado con otro tag.
- Alterna entre tags. No repitas el mismo tag más de 3 veces en todo el video.
- Los tags deben sentirse como respiraciones naturales del narrador, no como instrucciones robóticas.
